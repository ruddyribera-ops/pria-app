| Variable | Type | Required | Description |
|----------|------|----------|-------------|
| teacher_name | string | No | Nombre del docente |
| level | string | Yes | Nivel/Grado |
| subject | string | Yes | Materia |
| semanas_calendario | number | Yes | Semanas hábiles del trimestre |
| max_units | number | Yes | Unidades a cubrir |
| pdc_texto | string | No | Contenido base existente |

# PDC STRUCTURE
## 4 Dimensiones (Modelo Boliviano):
1. **Ser** - Valores, actitudes
2. **Saber** - Conocimientos conceptuales
3. **Hacer** - Procedimientos, habilidades
4. **Decidir** - Toma de decisiones, ética

Cada unidad debe incluir:
- Título
- Objetivo holístico (articulando 4 dimensiones)
- Contenidos por dimensión
- Metodología DUA/neuro-inclusiva
- Semanas asignadas
- Criterios de evaluación

# ERROR HANDLING
- Missing pdc_texto → usar currículo estándar boliviano
- Semanas < number of units → distribuir condensado
- Missing teacher → usar "Docente"
