DATOS DE ENTRADA:
- Docente: {teacher_name}
- Nivel / Grado: {level}
- Materia: {subject}
- Semanas hábiles del trimestre: {semanas_calendario}
- Unidades didácticas a cubrir: {max_units}

CONTENIDO BASE DEL PDC ACTUAL (extracto del docente):
{pdc_texto}

INSTRUCCIONES:
1. Genera exactamente {max_units} unidades didácticas para este trimestre.
2. Distribuye las {semanas_calendario} semanas hábiles equitativamente entre las {max_units} unidades.
3. Para cada unidad desarrolla:
   - Título de la unidad
   - Objetivo holístico (articulando las cuatro dimensiones: Ser, Saber, Hacer, Decidir)
   - Contenidos por dimensión
   - Metodología sugerida (incluyendo estrategias neuro-inclusivas y DUA cuando aplique)
   - Semanas asignadas (Ej: Semana 1 a Semana 4)
   - Criterios e instrumentos de evaluación
4. Al final, incluye una sección de "Observaciones del Docente" con espacio para notas.
5. Mantén coherencia con el contenido base del PDC subido por el docente.
6. Redacta en español, usando lenguaje pedagógico boliviano formal.
7. Estructura la respuesta con encabezados Markdown claros (##, ###) para cada unidad y sección.

REGLAS:
- NO generes JSON. La salida debe ser texto Markdown legible y descargable.
- NO incluyas saludos ni introducción innecesaria. Ve directo al PDC.
- Respeta estrictamente el número de unidades ({max_units}) y semanas ({semanas_calendario}).
- Si el contenido base del PDC está vacío o es insuficiente, genera el PDC basándote en el currículo estándar boliviano para {subject} en {level}.
